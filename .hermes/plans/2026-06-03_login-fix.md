# 登录无反应 — 根因与修复

## 根因
P5 安全加固将 `{ token }` 改为 `{ accessToken, refreshToken }`，
前端 `auth-context.tsx` 仍读 `data.token` → `undefined` → 静默失败。

## 修复
`auth-context.tsx` 三处改动：

1. localStorage key: `token` → `accessToken`
2. login() 读 `data.accessToken` 替代 `data.token`，同时存 `refreshToken`
3. register() 同样修正

## 风险
极低。纯前端字段名对齐，无需改后端。
