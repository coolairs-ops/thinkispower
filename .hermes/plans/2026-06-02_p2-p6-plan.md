## P6: 前端 SSR 优化 (~4h)

### 目标
Dashboard 和项目页改为服务端渲染，首屏 2s→0.5s

### 方案
渐进式改造，不影响现有功能：
- Dashboard: Server Component + 客户端加载 hook
- 项目页: 保留 'use client' 但加 loading.tsx
- 其余页面懒加载

### 步骤
1. `app/dashboard/page.tsx` → 拆为 Server + Client 两部分
   - Server: 静态骨架(标题/导航/空状态)
   - Client: `DashboardContent` (数据获取+交互)
2. `app/loading.tsx` → 全局 loading 骨架屏
3. `app/dashboard/loading.tsx` → Dashboard 骨架屏
4. `app/projects/[id]/loading.tsx` → 项目页骨架屏
5. 其余页面加 `dynamic(() => import(...), { ssr: false })` 懒加载

### 文件
- 修改: `app/dashboard/page.tsx`
- 新增: `app/loading.tsx`, `app/dashboard/loading.tsx`, `app/projects/[id]/loading.tsx`
- 新增: `components/skeleton-card.tsx`, `components/skeleton-page.tsx`
